# ВРАТА ХАОСА

### 1. КОНЦЕПЦИЯ ПРОЕКТА
**Название:** Gate of Chaos  
**Жанр:** Экшен-платформер с элементами головоломки  
**Целевая платформа:** PC (Steam), Nintendo Switch, PlayStation 5  
**Игроков:** 1  
**Режим:** Сюжетная кампания  

### 2. ОСНОВНОЙ ГЕЙМПЛЕЙ
#### 2.1. ЯДРО МЕХАНИКИ
- **Телепортационный контроль:** Игрок создает временные порталы на поверхностях
- **Пространственное манипулирование:** Перенаправление объектов через порталы
- **Динамическая физика:** Расчет траекторий при прохождении через порталы
- **Энергетический менеджмент:** Ограниченный заряд для создания порталов

#### 2.2. СИСТЕМА ПОРТАЛОВ
**Типы порталов:**
- **Синий портал:** Входная точка, можно создать на любой поверхности
- **Оранжевый портал:** Выходная точка, привязывается к созданному синему порталу
- **Стабильность порталов:** Портал исчезает через 15 секунд или при создании нового

**Ограничения:**
- Максимум 2 активных портала одновременно
- Нельзя создать портал на движущейся поверхности
- Некоторые материалы блокируют создание порталов

### 3. ПЕРСОНАЖ И СПОСОБНОСТИ
#### 3.1. ГЛАВНЫЙ ГЕРОЙ - ХРАНИТЕЛЬ ВРАТ
**Базовые способности:**
1. **Проекция Врат:** Создание пары порталов
2. **Фокус Времени:** Замедление на 70% на 3 секунды (перезарядка 30с)
3. **Импульсный Сдвиг:** Кратковременное ускорение в любом направлении
4. **Сканер Материй:** Подсветка портальных поверхностей

**Прогрессия:**
- Увеличение длительности порталов
- Сокращение времени перезарядки
- Увеличение максимальной дистанции между порталами

### 4. СТРУКТУРА УРОВНЕЙ
#### 4.1. АРХИТЕКТУРА
- **Секторы:** Изолированные комнаты с головоломками
- **Хабы:** Центральные зоны с доступом к 3-4 секторам
- **Босс-арены:** Крупные пространства с уникальной механикой

#### 4.2. ТИПЫ ИСПЫТАНИЙ
1. **Платформенные:** Точное использование порталов для преодоления препятствий
2. **Физические:** Перенос объектов через порталы для активации механизмов
3. **Тактические:** Уклонение от врагов через портальную телепортацию
4. **Скоростные:** Прохождение за ограниченное время

### 5. ВРАГИ И ПРЕПЯТСТВИЯ
#### 5.1. ТИПЫ ПРОТИВНИКОВ
- **Стражи Порядка:** Статические турели, стреляющие в прямолинейных лучах
- **Охотники Хаоса:** Мобильные враги, преследующие игрока
- **Подавители Порталов:** Создают зоны, блокирующие создание порталов
- **Поглотители Энергии:** Увеличивают перезарядку способностей

#### 5.2. ОПАСНЫЕ ЗОНЫ
- **Энергетические вихри:** Случайно телепортируют игрока
- **Нестабильная материя:** Взрывается при контакте с порталом
- **Временные аномалии:** Ускоряют или замедляют время локально
- **Антипортальные поля:** Полностью блокируют способности
